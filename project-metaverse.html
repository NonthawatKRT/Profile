SWU Metaverse

Type: Unity Game Development

Platform: Web / Multiplayer

Description: ได้รับรางวัลชนะเลิศในการประกวด Metaverse ของมหาวิทยาลัยศรีนครินทรวิโรฒ พัฒนาแพลตฟอร์ม Metaverse จำลองชีวิตในมหาวิทยาลัยให้สามารถเข้าถึงได้จากทุกที่ รองรับผู้เล่นหลายคน มีระบบล็อกอิน, สร้างและปรับแต่งอวตาร, และสำรวจพื้นที่ 3 มิติร่วมกับเพื่อน

My roles: รับผิดชอบพัฒนาระบบหลักทั้งหมด (Core Game Mechanics) เช่น ระบบผู้เล่น, ระบบ Multiplayer, ซิงโครไนซ์ GameObject และ Animation ข้ามเครือข่าย, ระบบ Teleport และการเชื่อมต่อพื้นที่ต่าง ๆ

Technologies Used:

  • Unity Engine
  • Blender (3D Modeling)
  • Mangos Framework (Core System)

Features:

  • Multiplayer login system
  • Custom avatar creation and upload
  • Realistic university simulation
  • Explore with friends in 3D environment
  • Networked animation and object synchronization
