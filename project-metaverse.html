SWU Metaverse

Type: Unity Game Development

Platform: Web / Multiplayer

This project is developed for Srinakharinwirot University as a visual metaverse platform. It simulates real university life online so anyone can access it from anywhere. This is a multiplayer environment with login system, avatar customization, and character exploration.

My roles: I handled Unity development including all game systems, player system, online networking, GameObject scripting, teleport system, and animation synchronization across the network.

Technologies Used:

  • Unity Engine
  • Blender (3D modeling)
  • Mangos Framework (core system)

Features:

  • Multiplayer login system
  • Custom avatar creation and upload
  • Realistic university simulation
  • Explore with friends in 3D environment
  • Networked animation and synchronization

Live Demo: You can try the demo at Mangosgo platform (may require a modern browser)
