NOS Insurance Web Application

Type: Full-Stack Web Application

Platform: Web / Customer & Employee

Description: พัฒนาเว็บแอปพลิเคชัน All-in-One Insurance Service สำหรับลูกค้า พนักงานขาย และผู้ดูแลระบบ ระบบนี้ครอบคลุมตั้งแต่การสมัครสมาชิก การเข้าสู่ระบบ การจัดการโปรไฟล์ การเลือกแพ็คเกจประกัน และระบบจำลองการชำระเงิน (Mock Payment) พร้อมการสร้าง QR Code

My roles: รับผิดชอบ Full-Stack Development: พัฒนาระบบ Backend และฐานข้อมูลทั้งหมด รวมถึง Frontend ของบางหน้า เช่น หน้า Register, Login, User Profile, Admin Dashboard และ Homepage ฟีเจอร์หลักที่พัฒนารวมถึง: ระบบลงทะเบียนพร้อม Email Verification, ระบบ Login แยกสิทธิ์ Customer / Sale / Admin, Admin Dashboard, การเชื่อมโยงลูกค้ากับพนักงานขายโดยอัตโนมัติ, ระบบ Profile Management, และระบบตะกร้าสินค้า / การชำระเงินแบบจำลอง

Technologies Used:

  • PHP, HTML, CSS, JavaScript
  • AppServ stack (Apache, MySQL)
  • MySQL Database Design

Features:

  • Registration system with email verification
  • Login system for customers, sales, and admin
  • Admin dashboard for managing sales and users
  • Automatic assignment of customers to sales
  • User profile system with editable data and profile picture upload
  • Mock payment system with QR code generation

Note: This project was developed as part of my classwork, so the demo is for testing in a local environment. Most features work normally, but some live demo issues may occur (e.g., registration may show errors while email verification still functions).
