NOS Insurance Web Application

Type: Full-Stack Web Application

Platform: Web / Customer & Employee

This project is an all-in-one insurance service platform for both customers and employees. It includes registration, login, profile management, insurance package selection, and a payment demo system.

My roles: Primarily backend development, including database setup, connection handling, and implementing all website features. I developed the registration system, email verification, forgot password functionality, login system for customers, sales, and admin, as well as admin dashboards. In addition, I implemented the front-end of several pages, including the user profile, admin dashboard, registration page, login page, and homepage.

Technologies Used:

  • PHP, HTML, CSS, JavaScript
  • AppServ stack (Apache, MySQL)
  • MySQL database design

Features:

  • Registration system with email verification
  • Login system for customers, sales, and admin
  • Admin dashboard for managing sales and users
  • Automatic assignment of customers to sales
  • User profile system with editable data and profile picture upload
  • Payment demo with QR code generation

Note: This is a 2-year project. Most features work normally, but some live demo issues may occur (e.g., registration may pop up errors while the email verification still works).
