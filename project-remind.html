RE-Mind Space

Type: Unity VR Game Development

Platform: PC / VR

Description: พัฒนาเกม Virtual Reality Multi-Sensory Game เพื่อช่วยฟื้นฟูความจำของผู้สูงอายุที่มีภาวะสมองเสื่อมระยะต้นและระยะกลาง เป็นความร่วมมือระหว่างคณะวิศวกรรมศาสตร์กับคณะกายภาพบำบัดของมหาวิทยาลัย

My roles: รับผิดชอบการพัฒนาระบบเกมทั้งหมด (All Game Logic Systems) รวมถึงระบบหลักของเกม (Main Game System), ระบบ Login, ระบบเก็บคะแนน (Player Score Store System) และฟังก์ชันของมินิเกมทั้ง 4:

  • เกมจับคู่ไพ่ (Card Matching Game)
  • เกมแยกของในครัวและตู้เย็น (Kitchen & Refrigerator Item Classification Game)
  • เกมแยกชนิดเสื้อผ้า (Dress Classification Game – เสื้อ, กางเกง, รองเท้า)
  • เกมจ่ายตลาด (Market Shopping Game – ผู้เล่นจดจำรายการสินค้า ใส่ตะกร้า และคำนวณราคา)

Technologies Used:

  • Unity Engine
  • Blender (3D modeling)
  • C# scripting for game logic
  • VR platform integration

Features:

  • Full VR multi-sensory experience
  • Login and score tracking system
  • 4 educational and memory-focused mini-games
  • Interactive environment for elderly players
  • All game logic implemented in Unity

Note: This project is a PC/VR game and cannot be run as a live demo online. The source code is also not publicly available due to licensing. If you would like to see a demo, please contact me.
