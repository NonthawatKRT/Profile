Educational Game Unity 3D Learning Media

EV Car Learning Project

November 15, 2025 - Ongoing • Educational Game Development

Project Overview

An educational Unity 3D game project focused on creating interactive learning media about electric vehicle (EV) systems and components. Developed as part of a collaborative team to make complex EV technology accessible and engaging through gamification.

This project aims to educate users about various EV subsystems, from battery technology and electric motors to charging systems and regenerative braking, through interactive 3D visualizations and gameplay mechanics.

My Role & Contributions

Served as Lead Unity Developer, responsible for directing the development team and implementing all core game systems on the Unity side. Key responsibilities included:

  • Led the Unity development team and coordinated technical implementation
  • Developed all core game systems and interactive learning mechanics
  • Designed the landscape map and interactive environment layout
  • Implemented interactive information display system with component highlighting
  • Created 3D visualizations and interactive models of EV components and systems
  • Developed user-friendly interfaces for exploring educational content

Learning Objectives

The game helps users understand EV technology through hands-on interaction, making technical concepts more approachable. Players can explore different vehicle systems, learn about energy efficiency, and understand the environmental impact of electric vehicles.
