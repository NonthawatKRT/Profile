Horror Game

June 13, 2024 • Unreal Engine Game Development

Project Overview

This horror game marks my first venture into Unreal Engine development. After mastering Unity with my previous project, I challenged myself to learn a new game engine and expand my technical skillset. This story-driven horror experience showcases my ability to adapt to new tools and technologies.

As a solo full-stack developer, I handled all aspects of development including level design, game design, programming, and optimization. This project was driven by pure passion and the desire to master Unreal Engine's powerful capabilities.

My Role - Full-Stack Developer

  • Level Design: Created immersive horror environments using Unreal's advanced rendering capabilities
  • Game Design: Designed gameplay mechanics, tension-building moments, and narrative flow
  • Unreal Programming: Implemented game systems using Blueprints and C++
  • Game Optimization: Leveraged Unreal's optimization tools to maintain high performance
  • Lighting & Post-Processing: Utilized Unreal's advanced lighting systems to create atmospheric horror scenes

Learning Journey

Transitioning from Unity to Unreal Engine presented exciting challenges and learning opportunities:

  • Mastered Unreal Engine 5's interface and workflow
  • Learned Blueprint visual scripting system alongside C++
  • Explored advanced rendering features like Lumen and Nanite
  • Discovered Unreal's powerful material editor and shader creation
  • Understood the differences in optimization approaches between engines

Development Highlights

This passion project expanded my technical capabilities significantly. Working with Unreal Engine taught me to think differently about game architecture and performance optimization. The engine's AAA-grade graphics capabilities allowed me to create more visually stunning and atmospheric horror experiences.

Building this project independently reinforced my problem-solving skills and ability to learn complex systems through documentation, community resources, and experimentation. The experience made me a more versatile game developer comfortable working with multiple game engines.
