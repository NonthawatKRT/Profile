About the Project

Developed an All-in-One Insurance Service web application for customers, sales staff, and administrators. The system covers registration, login, profile management, insurance package selection, and mock payment system with QR Code generation.

My Role

Responsible for Full-Stack Development: Developed the entire Backend and database system, as well as Frontend for several pages including Register, Login, User Profile, Admin Dashboard, and Homepage. Key features developed include: Registration system with Email Verification, Login system with role separation (Customer/Sale/Admin), Admin Dashboard, automatic customer-salesperson linking, Profile Management system, and shopping cart/mock payment system.

Platform

Website

Features

  • Registration system with email verification
  • Login system for customers, sales, and admin
  • Admin dashboard for managing sales and users
  • Automatic assignment of customers to sales
  • User profile system with editable data and profile picture upload
  • Mock payment system with QR code generation

Technologies Used

  • PHP, HTML, CSS, JavaScript
  • AppServ stack (Apache, MySQL)
  • MySQL Database Design
