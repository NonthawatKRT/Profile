About the Project

Hospital laboratory management application developed with Electron.js and Supabase to enhance convenience in sending test results, confirming test results, and quality control. The system helps doctors and medical technicians work more efficiently and quickly.

My Role

Backend Team Lead - Responsible for all Backend system development, including database design, Supabase integration, Authentication and Authorization management, automatic patient data retrieval system, and barcode data processing.

Platform

Desktop Application (Electron.js)

Features

  • Login and Forgot Password system
  • Role-based access control for Admin and Staff users
  • Barcode scanning system for patient identification
  • Database system with automatic patient data retrieval
  • Send and confirm laboratory test results
  • Quality Control system

Technologies Used

  • Electron.js - Desktop Application Framework
  • Supabase - Backend as a Service & Database
  • JavaScript/Node.js - Core Programming Language
  • PostgreSQL - Database (via Supabase)
