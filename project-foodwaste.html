Food Waste Manager

Type:Mobile Application (Flutter)

Platform: Android

This application assists small restaurants in managing their ingredients efficiently to minimize food waste. It calculates if there are enough ingredients for customer orders, automatically updates stock when an order is confirmed, allows adding new menu items, and manages overall inventory.

My roles: Full-stack development. I handled backend logic and database integration for all features, as well as front-end UI development to create a smooth user experience.

Technologies Used:

  • Flutter
  • SQLite Database
  • Firebase Authentication
  • Provider for state management

Features:

  • Inventory system
  • Order stock calculation
  • Automatic stock update when orders are confirmed
  • Add new menu items
  • User-friendly UI for Android
