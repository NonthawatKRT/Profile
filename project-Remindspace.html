About the Project

Developed a Virtual Reality Multi-Sensory Game to help restore memory for elderly patients with early to moderate stage dementia. A collaboration between the Faculty of Engineering and Faculty of Physical Therapy.

My Role

Responsible for developing all Game Logic Systems including the Main Game System, Login system, Player Score Storage System, and functions for all 4 mini-games:

  • Card Matching Game
  • Kitchen & Refrigerator Item Classification Game
  • Dress Classification Game (shirts, pants, shoes)
  • Market Shopping Game (players memorize shopping list, put items in basket, and calculate prices)

Platform

PC-VR

Features

  • VR Interaction System with Oculus Quest 2 Controllers
  • Login system with user profiles
  • Main game system with multiple mini-games
  • Player score storage and retrieval system

Technologies Used

  • Unity Engine
  • Oculus Integration SDK
  • Blender (3D Modeling)
  • C# Scripting
