Re-mind Space

Unity VR Game Development

About the Project

พัฒนาเกม Virtual Reality Multi-Sensory Game เพื่อช่วยฟื้นฟูความจำของผู้สูงอายุที่มีภาวะสมองเสื่อมระยะต้นและระยะกลาง เป็นความร่วมมือระหว่างคณะวิศวกรรมศาสตร์กับคณะกายภาพบำบัดของมหาวิทยาลัย

My role

รับผิดชอบการพัฒนาระบบเกมทั้งหมด (All Game Logic Systems) รวมถึงระบบหลักของเกม (Main Game System), ระบบ Login, ระบบเก็บคะแนน (Player Score Store System) และฟังก์ชันของมินิเกมทั้ง 4:

  • เกมจับคู่ไพ่ (Card Matching Game)
  • เกมแยกของในครัวและตู้เย็น (Kitchen & Refrigerator Item Classification Game)
  • เกมแยกชนิดเสื้อผ้า (Dress Classification Game – เสื้อ, กางเกง, รองเท้า)
  • เกมจ่ายตลาด (Market Shopping Game – ผู้เล่นจดจำรายการสินค้า ใส่ตะกร้า และคำนวณราคา)

Platform

PC-VR

Features

  • VR Interaction System with Oculus Quest 2 Controllers
  • Login system with user profiles
  • Main game system with multiple mini-games
  • Player score storage and retrieval system

Technologies Used

  • Unity Engine
  • Oculus Integration SDK
  • Blender (3D Modeling)
  • C# Scripting
